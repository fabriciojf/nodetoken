### Node Token
Simple example of WebToken generation in Node Js server with MongoDB

### Requirements
 - Node.js
 - MongoDB

### Install
Clone or Fork the project: 
```console
$ git clone https://github.com/fabriciojf/nodetoken.git
$ cd nodetoken
```

Run npm install
```console
$ npm install
```

Run the server
```console
$ node server.js
```
